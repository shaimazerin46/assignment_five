What is Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each part of the document (such as elements, attributes, and text) is represented as a node in the tree. This structure allows programs to interact with and manipulate the content, structure, and style of a web page dynamically.

how do you select element from DOM?

To select elements from the DOM, you can use various methods in JavaScript, depending on the type of element you want to select. These methods allow you to target elements based on their id, class, tag name, or other attributes.

How can you manipulate the DOM using JavaScript?

You can manipulate the DOM using JavaScript to dynamically change the structure, content, and style of web pages.

What is the difference between querySelector() and querySelectorAll()?

querySelector() selects the first matching element and returns a single element or null if none is found. querySelectorAll() selects all matching elements and returns a NodeList, which can be empty if no matches exist. Use querySelector() for single elements and querySelectorAll() for multiple elements.
